### Project Overview
Bellabeat is a high-tech company that develops smart wellness devices designed to help women monitor activity, sleep, and overall health.
In this project, I worked as a Junior Data Analyst supporting the marketing team by analyzing smart device usage data to understand user behavior and generate insights for marketing strategy.

🔗 [View Full Analysis](https://medium.com/@bintangpradanaa/optimizing-bellabeats-marketing-strategy-from-smart-device-usage-trends-d54638c7ad7d)

### Objective
- Identify user behavior patterns from smart device data  
- Understand the relationship between activity, sleep, and health metrics  
- Provide actionable insights to support marketing decisions  

### Approach
- Data cleaning and preprocessing (handling missing values, data types, merging datasets)  
- Feature engineering (e.g., day of week, hourly activity)  
- Exploratory data analysis and trend analysis  
- Correlation and behavioral analysis

### Key Insights
- Activity is mostly light, peaking on Saturdays and dropping on Sundays (weekend behavior shift)  
- Steps are relatively stable, while calories and distance fluctuate more  
- Higher activity levels directly increase calories burned  
- More active users tend to have better sleep quality
- Sleep patterns are inconsistent, with oversleeping on weekends  
- Most users are in the normal to overweight BMI range
- Manual tracking is frequent, indicating engagement but potential data issues  

### Recommendations
#### 1. Enhance User Activity
- Introduce weekend-focused challenges (e.g., Sunday step goals) to address low activity periods  
- Implement gamification features such as points, badges, and leaderboards to increase engagement  
- Send contextual reminders (morning & afternoon) to encourage light physical activities  
- Provide guided content (e.g., stretching, short exercises) to support daily activity  

#### 2. Improve Sleep Quality
- Provide personalized sleep reminders and pre-sleep routine guidance  
- Develop sleep insights dashboards to help users track and understand sleep patterns  
- Encourage consistency through incentives (e.g., badges for good sleep habits)  

#### 3. Marketing & Engagement Strategy
- Align campaigns with user behavior patterns (e.g., weekend promotions & activity trends)  
- Offer product bundles combining fitness and sleep-related products  
- Leverage storytelling (testimonials, educational content) to increase user trust and engagement  

#### 4. Optimize Data Accuracy
- Improve automated tracking to reduce reliance on manual input  
- Implement validation systems to detect inconsistencies in user-reported data  
- Educate users on accurate data reporting through in-app guidance
